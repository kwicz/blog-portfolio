# dark-autumn-portfolio

Archived snapshot of the original blog/portfolio theme as of 2026-05-25.

## What's here
- `app/` — all page and component templates
- `global.css` — CSS variables, base styles
- `tailwind.config.js` — custom color tokens (ivory, slate, rose, gold, sage, ocean), fonts, animations
- `contentlayer.config.js` — content model definitions

## Theme character
Dark navy + warm ivory palette. Serif display fonts (Poppins/Raleway). Particle animation on home. Card hover effects. Dark mode toggle. Minimal, editorial feel.

## Pages
- **Home** (`app/page.tsx`) — hero with name, title, animated particles
- **Posts** (`app/posts/`) — blog listing with search + individual post pages
- **Projects** (`app/projects/`) — project grid with category filter + individual project pages
- **Contact** (`app/contact/`) — contact page
