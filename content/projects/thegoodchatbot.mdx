The Good Chatbot is an AI-powered WordPress plugin that helps website visitors quickly discover relevant articles and content. It uses Anthropic’s Claude AI to provide natural, conversational responses that guide users to the information they’re looking for.

I built this tool after realizing how expensive most chatbot solutions are. It was designed as a custom build for my company, specifically to support digital agencies that want to improve user experience and content discovery on their clients’ websites.

This was my first foray into creating generative AI tools, and it was a super fun project to make.

## Features

- AI-powered content recommendations with Claude
- Seamless WordPress integration
- Customizable to match client branding
- Built to improve content discovery and user engagement

## Tech Stack

**Tech Stack:** WordPress, PHP, JavaScript, Anthropic Claude API

## Project Story

Building The Good Chatbot gave me a hands-on opportunity to explore generative AI. I wanted to create something practical and affordable that could directly improve the work of digital agencies. Seeing it in action as a plugin for WordPress made the project especially rewarding, since it combined new AI technology with a platform widely used by clients.
