Snap Insights is a Chrome browser extension built for UX researchers. It allows you to quickly capture screenshots, add contextual annotations, and record voice notes to preserve insights in real time.

I built this project because I work at a UX agency and saw how useful it would be for researchers who need to document their findings clearly and efficiently. I created the design, icons, and logo myself—and I think it turned out really cute.

## Features

- Capture and save annotated screenshots
- Add contextual notes directly on the page
- Record voice insights alongside visuals
- Lightweight and easy to use in research sessions

## Tech Stack

**Tech Stack:** React, TypeScript, Chrome Extensions API

## Project Story

This project grew out of my day-to-day work in UX research, where teams often juggle multiple tools to capture insights. I wanted to create something simple and unified that would streamline that process. Designing the icons and logo was especially fun—it gave me the chance to make the extension both functional and visually inviting.
