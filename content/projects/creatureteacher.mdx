### About the Project

Creature Teacher is an educational web application designed to teach students about DNA and codons through interactive testing. Inspired by Pokémon, this project brings biology concepts to life with engaging creature designs and a fun learning experience.

### The Team

This project was a collaborative effort between a team of passionate developers with diverse backgrounds:

Katy (Former Teacher) - Designed the learning framework and structured the educational approach.

Biologist - Provided scientific accuracy and ensured the DNA/codon concepts were well represented.

Graphic Designer - Created adorable creature visuals to make the experience engaging.

Backend Developer - Built a solid foundation for handling game logic and database integration.

Together, we combined our expertise to create a well-rounded and immersive learning experience.

### Features

Interactive Testing - Students can experiment with DNA sequences and learn how they translate into creatures.

Dynamic Creature Generation - Based on codon sequences, different creatures are generated.

Gamified Learning - Inspired by Pokémon, the app makes genetics fun and accessible.

### Status & Future Goals

While Creature Teacher isn't successfully deployed yet, we are excited to revisit and refine the project. The dream is to see these cute little creatures go live again and help students engage with science in a creative way.

Stay tuned for future updates! 🚀
